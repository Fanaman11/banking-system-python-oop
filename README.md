# ATM Transactions Data Pipeline (Python Project)

## Overview

This project is a basic ATM simulation built using Python. It allows users to perform simple banking operations like checking balance, depositing money, and withdrawing money.

The project also demonstrates a simple data flow where user input is processed and stored, similar to a basic data pipeline.

---

## Features

* Check account balance
* Deposit money
* Withdraw money
* Store transaction history
* Input validation for user entries

---

## Technologies Used

* Python

---

## How It Works

The program follows a simple flow:

1. User selects an option from the menu
2. The program processes the request
3. The balance is updated accordingly
4. Transactions are stored in a list
5. Results are displayed to the user

---

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/fanaman11/ATM-Transactions-Data-Pipeline.git
   ```

2. Go into the project folder:

   ```bash
   cd ATM-Transactions-Data-Pipeline
   ```

3. Run the program:

   ```bash
   python atm.py
   ```
